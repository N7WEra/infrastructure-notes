# FreeBSD

find freebsd version:

`uname -a`

view installed software:

`pkg_info` or `pkg info`

view passwords:

`vipw` or `cat /etc/master.passwd`

View listening ports:

`netstat -an -p tcp`

view services:

`ps aux`

Shows operating system info

`uname -a`

search files for password:

`grep -ri 'password' *`
